## Key Takeaways and Conclusions
- Formato di immagine basato su XML utilizzato per definire grafica vettoriale bidimensionale. A differenza delle immagini raster (come JPEG o PNG), che sono costituite da una griglia di pixel, le immagini SVG sono composte da forme geometriche come linee, cerchi, rettangoli e testo, definite attraverso descrizioni matematiche.
