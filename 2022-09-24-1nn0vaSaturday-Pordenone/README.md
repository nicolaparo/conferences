# Realtime app con Blazor ed Azure

## English

### Presentation Overview
In this session we explore how to build a realtime application with Blazor WebAssembly and Azure Web PubSub, then deploy it on Azure Static Web Apps.


### Key Takeaways
- The session explains how to frame Realtime app con Blazor ed Azure and move from concept to execution.
- It maps technology and design choices to practical scenarios around Sabato 24 settembre 2022.
- It provides reusable guidance and trade-offs for teams working on Cloud Solutions Engineer in beanTech.
